### 4. 系统功能划分

业务功能划分初期按照微博功能进行学习，现简单划分如下：

1. 用户系统：包括新用户的注册、登录、注销，对用户的登录使用SSO设计
2. 博文系统：包括用户对博文的发布、删除、点赞、转发、评论（互动），博文的内容包含文字以及图片，
3. 热搜系统：参考新浪微博的热搜实现进行简单复现，由用户博文的受欢迎程度进行排名展现
